## 📖 Overview

This project is a modern take on the classic web calculator. Beyond standard arithmetic operations, it focuses heavily on **User Experience (UX)** and **User Interface (UI)** design. The application utilizes a glassmorphism aesthetic (backdrop blurs and translucent panels), a smooth infinite-scrolling marquee, and dynamic scroll-triggered footer animations to create a premium feel. 

## 🚀 Features

- 🧮 **Functional Arithmetic:** Handles standard math operations seamlessly.
- ⌨️ **Full Keyboard Support:** Users can type equations directly using their numpad or keyboard, complete with an elegant Toast notification reminder.
- ✨ **Advanced Animations:** Includes an infinite horizontal scrolling quote banner and a smooth, scrubbed footer reveal powered by GSAP & ScrollTrigger.
- 🎨 **Glassmorphism UI:** Built with Tailwind CSS to create a beautiful, frosted-glass interface over a custom background.
- 📱 **Responsive Design:** Ensures a great experience across different screen sizes.
